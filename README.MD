# Ad Security Checker

## Overview

Ad Security Checker is an AI-assisted cybersecurity application designed to help users evaluate the trustworthiness of advertisement URLs and websites by analyzing multiple security indicators.

The objective is to provide an evidence-based risk assessment instead of simply labeling a website as "safe" or "unsafe."

---

## Project Goals

The project will help users:

- Validate URLs
- Detect suspicious domains
- Identify possible brand impersonation
- Evaluate domain reputation
- Analyze redirects
- Calculate a security risk score
- Explain why a website appears trustworthy or suspicious
